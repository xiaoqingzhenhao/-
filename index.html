Flexbox 布局学习笔记

Flexbox 简介

Flexbox（弹性盒子布局）是一种CSS布局模式，它提供了一种高效的方式来布局、对齐和分配容器内项目的空间，即使它们的大小未知或动态变化。

Flexbox 容器属性

  • display: flex; - 启用 Flexbox 布局。
  • flex-direction: - 控制主轴方向（row, row-reverse, column, column-reverse）。
  • flex-wrap: - 控制子元素是否换行（nowrap, wrap, wrap-reverse）。
  • justify-content: - 控制子元素在主轴上的对齐方式（flex-start, flex-end, center, space-between, space-around, space-evenly）。
  • align-items: - 控制子元素在交叉轴上的对齐方式（flex-start, flex-end, center, baseline, stretch）。

Flexbox 项目属性

  • order: - 控制子元素的排序（数值越小，越靠前）。
  • flex-grow: - 定义子元素的放大比例。
  • flex-shrink: - 定义子元素的缩小比例。
  • flex-basis: - 定义子元素的默认大小。
  • align-self: - 允许子元素有与其他元素不同的对齐方式。

盒子变形

盒子变形通常指的是改变元素的尺寸和位置，这可以通过CSS的 transform 属性实现。例如，:hover 伪类可以触发变形效果。

动画

动画可以通过CSS的 @keyframes 规则和 animation 属性来创建。

变形和动画示例

看我变形！
<style>
  .flex-item:hover {
    transform: scale(1.1) rotate(10deg);
  }
  
  @keyframes example {
    0% { background-color: #f4f4f4; }
    100% { background-color: #6495ED; }
  }
  
  .flex-item {
    animation: example 4s infinite alternate;
  }
</style>
